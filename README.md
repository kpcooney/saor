# Saor

Saor — an AI-powered SDLC orchestration platform. Coordinates specialist agents through requirements, UX, architecture, implementation, testing, and deployment using scoped agent identities, a shared memory layer, reference-based handoffs, and a full audit trail.

Built with Tauri 2.0 (Rust backend), Svelte + TypeScript (frontend), and the Claude Agent SDK (TypeScript agent layer).
