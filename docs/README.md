# docs/

This directory holds all project-level documentation for Saor. The authoritative reference for technical decisions is the architecture research document at [`architecture/sdlc-agent-architecture-research-v4.md`](architecture/sdlc-agent-architecture-research-v4.md) — read it before making structural decisions, particularly Section 5 (Reference-Based Handoff Protocol), which is the novel core of the architecture.

Beyond the architecture document, this directory contains Architectural Decision Records (ADRs) in `adr/` — short documents capturing significant decisions, the options considered, and the chosen rationale. Any other project-level documentation (research notes, design explorations) lives here as well.
